## 📖 它能做什么？

### 🤖 四大AI模型协同工作

慧眼选鸟采用多个专业AI模型，从多个维度评估您的鸟类照片：

| 功能模块 | 作用 |
|---------|------|
| **YOLO11 检测引擎** | 精准识别照片中的鸟类位置 |
| **SuperEyes 鸟眼模型** | 检测眼睛是否清晰可见 |
| **SuperFlier 飞鸟模型** | 识别飞行姿态，给飞版加分 |
| **TOPIQ 美学评分** | 评估画面构图和整体美感 |

---

### ⭐ 智能0-3星评分系统

软件会根据 **头部锐度** 和 **美学评分** 两大核心指标，自动为每张照片评定星级：

| 星级 | 评判标准 | 建议 |
|------|----------|------|
| ⭐⭐⭐ 三星 | 锐度+美学双达标 | 🌟 精品！值得后期精修 |
| ⭐⭐ 二星 | 锐度或美学达标其一 | 👍 不错，可考虑保留 |
| ⭐ 一星 | 有鸟但指标都未达标 | 🤔 一般，可以删除 |
| 0 零星 | 无鸟或质量太差 | ❌ 建议直接删除 |

---

### 🏷️ 三大特殊标记

**1️⃣ Pick 精选标记**
> 从所有三星照片中，选出锐度和美学 **双排名前25%** 的交集，自动标记为"精选"——这些就是您这批照片中的精华！

**2️⃣ 飞鸟绿标 🟢**  
> AI识别到飞行姿态的照片会用绿色标签标记，并获得额外加分。飞版难拍，理应获得更多认可！

**3️⃣ 曝光检测【可选】**
> 可开启曝光问题检测，过曝或欠曝的照片会自动降星处理。（默认关闭，需手动开启）

---

## 📂 自动整理功能

处理完成后，软件会自动将照片 **按星级分类** 到不同文件夹：

```
📁 您的照片文件夹
 ├── 📂 3_stars/  ← 三星精品
 ├── 📂 2_stars/  ← 二星良品
 ├── 📂 1_stars/  ← 一星普通
 └── 📂 0_stars/  ← 零星淘汰
```

## 🚀 使用方法（超级简单！）

### 第一步：选择文件夹
将包含鸟类照片的文件夹**拖入**软件窗口，或点击"浏览"选择

### 第二步：调整参数（可选）
- 锐度阈值：建议200-600之间
- 美学阈值：建议4.0-7.0之间
- 可开启/关闭飞鸟检测、曝光检测

### 第三步：开始处理
点击"开始处理"按钮，等待AI完成分析

### 第四步：查看结果
- 照片自动分类到星级文件夹
- 导入Lightroom即可使用

## 👁️ 为什么眼睛这么重要？

**眼睛是鸟类摄影的灵魂！**

慧眼选鸟特别关注鸟眼的可见度，评分逻辑如下：

- 👁️ 眼睛清晰可见 → 正常评分
- 👁️ 眼睛略微模糊 → 最高2星
- 👁️ 眼睛完全不可见 → 最高1星

这与我们摄影师的审美标准完全一致！
